===========================================================
  FIREBASE SETUP - google-services.json REQUIRED
===========================================================

This app uses Firebase Cloud Messaging (FCM) for push
notifications. The Android build WILL FAIL without the
google-services.json file.

HOW TO GET IT:
1. Go to https://console.firebase.google.com/
2. Open your project (or create one)
3. Click the gear icon → "Project Settings"
4. Under "Your apps", find the Android app
   (or click "Add app" → Android)
5. Use package name: com.reddifleadmonitor
6. Download google-services.json
7. Place it HERE in: android/app/google-services.json

NOTE: The file at Reddif/backend/firebase-service-account.json
is the SERVER-SIDE key (for sending notifications from your
backend). It is NOT the same as google-services.json which
the Android app needs.
